# The Borderland Map

https://www.theborderland.se/map/

## Available commands

## Serve for development

```bash
npm ci
npm run serve
```

## Build for production

```bash
npm run ci
npm run build
```

All the production assets can then be found in the folder `public`, can be deployed to any web server for static files
